Boolean selection

You can apply comparison operators between an array and a number or between two arrays.

The comparison is applied to every element in the array

The return will be an array of true and false


You can use np_array[np_array>5] to get all elements in np_array that are greater than 5.
You can use & (and), | (or) to combine multiple comparison operators.
For instance, np_array[(np_array<1)|(np_array>8)] will return all elements smaller than 1 or greater than 8.
