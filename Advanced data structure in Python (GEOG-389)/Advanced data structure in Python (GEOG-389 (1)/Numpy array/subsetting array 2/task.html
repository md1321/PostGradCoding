Subsetting array 2

You can get a subset of an array using np_array[i:j][m:n], which returns the part of the array from the (i+1)th to jth row, and from (m+1)th to nth column.

You can also omit the starting index to get elements until the ending index

Analogously, you can also omit the ending index to get elements from the starting index

Let's try some examples to understand how they work



You can use indices in [] to access individual element in an array. For instance, np_array[a][b] will return the element at the (a+1) row and (b+1) column.
You can use : to get an entire row or column. For instance, np_array[1,:] will return the entire 2nd row.
