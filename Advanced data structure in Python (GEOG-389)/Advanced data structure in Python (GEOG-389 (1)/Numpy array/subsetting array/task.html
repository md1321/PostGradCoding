Subsetting array

The indexing and subsetting of numpy array is the same as list. You can use index in [] to get individual elements or subset of elements from an array



You can use indices in [] to access individual element in an array. For instance, np_array[a][b] will return the element at the (a+1)th row and (b+1)th column.
You can use : to get an entire row or column. For instance, np_array[1,:] will return the entire 2nd row.
