Subsetting data frame

In a data frame, you can select rows according to the values in specific columns.

You can right-click on election.csv in the file list (in the left panel) -> Copy Path to copy the full path of the file.
Use df[df.colname>3] to select rows in which the value in 'colname' is greater than 3.
df[df['colname']>3] does the same.
data frame is essentially built on a 2D numpy array. Thus, you can use df.shape to get the shape (dimension) of a data frame.
df['column'].sum() will return the sum of numbers in 'column'. But this only applies to columns of numbers (integer, float, double), not string columns.
