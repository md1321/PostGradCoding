Read a dataframe from a csv file

In this task, you will learn to manipulate data frame, which is a data structure designed to store tables and spreadsheets.

You will read a csv file stored in the same folder of the python file.

The spreadsheet contains data about presidential election 2012 in Pennsylvania. Try to understand the meaning of the columns by the column names.

The meaning of the columns:

- county: county

- state: state

- total: total votes received

- Obama: percentage of vote for Obama

- Romney: percentage of vote for Romney

- winner: winner in the county

- voters: total eligible voters in the county.

- turnout: percentage of eligible voters who actually voted

- margin: percentage of eligible vote who didn't vote.

You can use df[a:b] to get (a+1)th to bth rows of df.
You can get columns in a data frame using two different methods,i.e. df['colname'] or df.colname
You can get multiple columns and group them to a new data frame. For instance, new_df=df[['col1','col2','col3']]
