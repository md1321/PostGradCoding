Selecting data by attributes

An vector dataset read by geopandas is a special types of data frame, which is called geodataframe.

A geodataframe include geometries of the spatial locations and attributes.

You can use the subsetting operations for dataframe to select features (rows) in geodataframe as well.

In this task, you will select two different types of crimes, plot them in two maps and compare their spatial distributions


You can use data[data['ColName'] == something] to select features (rows) according to the values in a column
